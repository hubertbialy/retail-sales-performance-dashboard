# Retail Sales Performance Dashboard – Excel

## Project Overview

Interactive sales analytics dashboard built in Microsoft Excel.

The project analyzes retail transaction data and provides insights into sales performance, profitability, customer behavior, and operational efficiency.

The dataset includes order-level transactional data with product, region, shipping, and customer attributes.

---

## Key KPIs

- Total Orders: 2,400
- Units Sold: 11,997
- Revenue: 649,020
- Average Rating: 3.96
- Average Delivery Time: 2.34 days

---

## Analytical Scope

### 1. Sales & Revenue Analysis
- Weekly trend (Last 13 weeks: Quantity vs Amount)
- Revenue tracking
- Quantity purchased per order
- Sales distribution by product category
- Product-level performance ranking

### 2. Customer Insights
- Customer distribution by region (California map)
- Product popularity by gender
- Customer satisfaction analysis
- Customer purchase channel preference (App, Website, Instagram, etc.)

### 3. Operational Performance
- Shipping time distribution
- Order mode comparison
- Delivery performance metrics
- Channel performance analysis

---

## Dashboard Features

- Interactive slicers:
  - Order Mode

- Pivot Tables & Pivot Charts
- KPI summary cards
- Heatmap (channel vs gender preference)

---

## Business Use Case

This dashboard simulates a retail sales controlling scenario and supports:

- Revenue monitoring
- Channel performance evaluation
- Customer segmentation analysis
- Operational efficiency assessment
- Product demand optimization
- Delivery time performance tracking

---

## Example Insights

- Website and App channels generate the majority of revenue.
- Female customers represent the largest purchasing group.
- Most orders contain 2–3 items.
- Shipping time is concentrated between 1–3 days.
- Certain product categories show high volume but moderate revenue contribution.
